# Public packages

**Add the packages to your project**

1. Open a terminal and `cd` to your project directory root (where `pubspec.yaml` is located).
```bash
cd your/project/directory/root
```

2. Add the repository to your project.
```bash
dart pub add http --git-url=https://github.com/flutterfocus/ff-packages--git-ref=main
```
